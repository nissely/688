## 心得

---
隨著數位與分享經濟的時代來臨，加上智慧城市朝向可持續性發展等大環境趨勢下，交通行動服務系統因應而生。交通行動服務是指將各種運輸服務，如公車、汽車與自行車的共享模式。
那所謂的訂閱式移動服務就是不用一次性投入大筆資金，省去處理中古車輛的麻煩
只要按月支付租金，就能用小錢開好車，訂閱期間還可輕鬆換車
不用被同一台車綁住您的生活，享受更輕鬆自在的用車服務而訂閱機制能夠更客制化的方式來增添個人需要的方案、內容，就和現代社會的火車機票相同，需要多少則使用多少，同時訂閱也可以更了解使用此階層的族群喜好、年齡來做個人化的設定
透過赫爾辛基與阿姆斯特丹的發展案例中，發現透過大數據分析後國家型政策計畫、大型基礎建設的改善與調整與公私合作之夥伴關係，並整合公共運輸系統也可以和現今各方面的Pay來更快速的完成支付動作，減少時間的支出，再來也會開啟更多的市場消費，像是隨處可見的Uber也已經開始朝這方面跟進，漸漸的「空中計程車」也會進入我們生活周遭的每個角落，讓生活得以更方便的方式來運行。
